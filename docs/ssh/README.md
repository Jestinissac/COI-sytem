# SSH

Documentation for SSH access, deployment, and server connections related to Envision PRMS / COI.

**Do not commit:**
- Private keys (`id_rsa`, `id_ed25519`, etc.) — keep these in `~/.ssh/` on your machine only.
- Passwords or secrets — use environment variables or a secrets manager.

**You can add here:**
- How to connect to dev/staging/production servers
- Server hostnames and roles (e.g. app server, DB server)
- Deploy or backup steps that use SSH
- Links to internal wikis or runbooks
