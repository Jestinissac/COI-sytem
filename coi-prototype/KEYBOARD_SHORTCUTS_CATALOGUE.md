## Sequence Shortcuts Explained

Sequence shortcuts require pressing keys in order within 1 second.

### Example: `G` then `O`

1. Press `G` - System starts sequence buffer
2. Press `O` within 1 second - System matches "G O" sequence
3. Action executes (navigates to Overview)
4. Sequence buffer clears

**Timeout:** If you don't press the second key within 1 second, the sequence resets.

## Usage Tips

### 1. **Learn the Basics First**
Start with these essential shortcuts:
- `Ctrl/Cmd + K` - Search
- `Ctrl/Cmd + /` - Help
- `G` then `O` - Go to Overview
- `Esc` - Close modals

### 2. **Use Sequences for Navigation**
Instead of clicking through tabs:
- `G` then `E` - Execution Queue
- `G` then `M` - Monitoring
- `G` then `R` - Renewals

### 3. **Search is Your Friend**
Use `Ctrl/Cmd + K` to quickly find:
- Requests by ID
- Clients by name
- Configuration pages
- Any page in the system

### 4. **Keyboard-First Workflow**
Try to complete common tasks using only keyboard:
1. `Ctrl/Cmd + K` - Open search
2. Type what you need
3. `Enter` - Select result
4. Navigate with `Tab` and `Enter`

## Troubleshooting

### Shortcuts Not Working?

1. **Check if you're typing**
   - Shortcuts are disabled when typing in input fields
   - Click outside input fields to activate shortcuts

2. **Check your platform**
   - Mac: Use `âŒ˜` (Command) key
   - Windows/Linux: Use `Ctrl` key

3. **Check sequence timing**
   - Sequence shortcuts must be pressed within 1 second
   - Try again if you missed the timing

4. **Check browser shortcuts**
   - Some browsers override certain shortcuts
   - Try a different key combination

5. **Refresh the page**
   - If shortcuts stop working, refresh the page
   - This re-initializes the shortcut system
