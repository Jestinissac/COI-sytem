## Color Scheme (BDO-Inspired)

### Primary Colors
- **Primary Blue:** `#0066CC` - Main actions, headers
- **Accent Red:** `#CC0000` - Critical actions, errors
- **Gradients:** Used throughout for depth and visual interest

### Section Header Colors
- Requestor: Primary blue gradient
- Document: Blue gradient
- Client: Green gradient
- Service: Purple gradient
- Ownership: Orange gradient
- Signatories: Indigo gradient
- International: Teal gradient

### Status Colors
- Draft: Yellow
- Pending: Blue
- Approved: Green
- Active: Primary blue
- Rejected: Red

## Visual Enhancements

### Shadows & Depth
- **Cards:** shadow-lg (enhanced from shadow-card)
- **Hover:** shadow-xl, shadow-2xl
- **Buttons:** shadow-lg with hover:shadow-xl

### Borders
- **Inputs:** border-2 (thicker, more visible)
- **Cards:** border-2 with hover color changes
- **Focus:** border-primary-500 with ring-2 ring-primary-100

### Spacing
- Increased padding (p-6, p-8 instead of p-4)
- Better gaps (gap-6, gap-8 instead of gap-4)
- More breathing room between sections

### Typography
- Larger headings (text-3xl, text-5xl)
- Font weights: font-semibold, font-bold
- Better text hierarchy

### Animations & Transitions
- Hover effects: transform hover:-translate-y-2 (lift)
- Smooth transitions: transition-all
- Focus states: ring effects
- Button hover: gradient shifts

## Design Principles Applied

1. **Visual Hierarchy** - Clear section separation with color-coded headers
2. **Consistency** - BDO-inspired colors throughout
3. **Feedback** - Hover effects, focus states, transitions
4. **Professional** - Maintains corporate look while being modern
5. **Accessibility** - Good contrast, clear labels, focus indicators
