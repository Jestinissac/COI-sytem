## Sub-Categories (Business/Asset Valuation)

The following sub-categories are available for Business/Asset Valuation services:
- **Acquisition**
- **Capital Increase**
- **Financial Facilities**

These are stored in the `service_type_categories` table and can be selected when choosing "Business Valuation" or "Asset Valuation" as the service type.

---

## API Endpoint

**GET** `/api/integration/service-types`

Returns the complete service type list organized by category, with sub-categories for Business/Asset Valuation services.
