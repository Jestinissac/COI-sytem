# COI System

Governance, scope, rules, and research for the Conflict of Interest (COI) system. This folder consolidates the former **"COI System"** content.

## Contents

- **Governance & framework** – COI Governance, Integrated/Project Resource Management, Safeguarding Conflict Integrity.
- **CMA / IESBA** – CMA matrix, IESBA mapping, rule builder changes, phase verification.
- **Scope & plans** – Prototype scope, prototype plan, data seeding, user journeys.
- **Research** – Architecture decisions, event-driven design, fuzzy matching, state/HRMS integration.
- **Q&A/** – Implementation decisions, journey gaps, critical questions.
- **extracted_text/** – Extracted text from source documents.

Use for production design and compliance; implementation logs stay in `coi-prototype/docs/prototype/`.
